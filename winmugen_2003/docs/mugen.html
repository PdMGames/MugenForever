Overview

    M.U.G.E.N is a 2D fighting game engine that lets you create your own characters and stages. Oriented more towards flexibility, making characters with M.U.G.E.N may initially seem a daunting task, but once you understand the basics, learning the rest of the tricks should come naturally.

    Some of the documentation here is available in .rtf format as well as .html. The .rtf versions are more suitable for printing. To view the .html files, we recommend you use a browser that supports style sheets.
